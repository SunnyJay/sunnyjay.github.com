开篇

时间过的太久，记忆模糊，也不知道有多少年没有正经的写点文字了。

其实严格来讲，或许要追溯到高考的那篇作文吧，自从当年交了那份卷子以后，似乎就开始和所谓的“语文、文艺、文学”越离越远，如同相交线越过交点，一交不可收拾。

尽管高中的时候我自认为在语文这个科目上还有些“想法”，但事实是我已与所谓的文艺青年道别，并逐渐由普通青年蜕变成了二逼青年。

为什么要写博客

今天写这篇，当然不是突发奇想，这是我建立博客的开篇之章，但为什么突然想写博客呢，总结了下，大概原因有下面几个：

  • 一直以来，我都渴望拥有自己的个人网站，当然我也一直在努力和实践，从高中开始就有属于自己的网站，上大学以后也建立了并维护了几个，但是都因为各种各样的原因没有坚持下来。现在上了班，有了稳定的工作和收入来源，闲暇时间，我想继续自己的这份追求，拥有一个长期稳定的、美观的、可以学习交流并展现自己的平台。这里额外提一下，作为纪念，我将在未来的某个时间专门写一篇关于这方面过往的博文，也算祭奠我“那些年追过的网站”。

  • 最近在公司研究 ELK 框架，在谷歌查找相关资料时，无意中发现作者 wklken 的博客，里面除了技术分享之外，还有很多作者个人对生活、职业、读书等方面的理解，让我启发甚多，引起了我很大的共鸣。尤其是看到了这篇文章《我为什么要写博客》，我思绪万千。我很早之前就开始接触云笔记产品，至今里面记录的各式各样笔记已经 2000 多份，但是原创与引用杂乱不堪，质量参差不齐，尤其是专业技术方面的笔记，个人对知识的理解和记录非常混乱，那我为什么不能把自己的见解搬到博客上分享给大家呢，正如前面博文里所说能更好的梳理自己的思路，还能有与大家互相沟通学习的机会。

  • 也就是前几天，公司的一位同事，也是我很好的朋友，上班之余，无论国家政策、社会现状、IT 技术、伤风低俗之类我们都无话不谈，但是我却一直不知道他是一位低调却很有才华的人。当我进入他的博客阅读了几篇他的博文之后，让我震撼，打心底敬佩，因为我无法相信这是一位 IT 技术男的所作所为。我思考，同样作为理工科男，同样从事于 IT 行业，我们为何有这样的差距。当然，毕竟是我的朋友，我不打算把他的博客放在这里。

差不多就是这几个原因，让我有了建立博客的想法。

建立博客

因为之前有过 Web 开发和 Python 基础，经过简单的调研和学习，我使用 Pelican 去搭建自己的博客，访问地址:http://sunnyjay.github.com。

目前网站暂时挂载到 github上，域名也使用 github 二级域名，等到博客建立较为完善时，我准备先将其转移到云主机上（目前打算移转移到新浪云上）维护，并使用自己专属的顶级域名。

对于博客的搭建过程和方法，未来我也会写成博文记录下来，作为我的知识分享，与大家互相交流学习。

后话

说实话，其实真的很想写些文字，不夸张的说，憋了估计有好几年，尤其是上了研究生之后，深深陷入自己的理工科世界。

基本上，三年生涯没有读过什么文学作品，著作更谈不上了，将时间都花费到了自认为有价值的领域，主要都是计算机专业相关知识的学习，有段时间还沉溺在了游戏之中，我想这也许是很多 IT 宅男的共同特点吧。

但是，时间久了，就真的越来越能体会自己内心深处的某种不适，这种感觉我不知道如何表达，或许是一种归属感的愈发缺失吧。

我自认为自己是一个比较闷骚的人，但却也一直想用文字去表达自己的内心情感，当然曾经那些故作深沉矫、揉造作的、经常充满负能量的空间心情状态不算。

现在，我希望能利用这个博客，真正开启自己对生活、工作、学习等新的认知和理解。

同时，从现在开始，一定要多读书，不能仅局限于专业书了，我要用书籍打开自己的世界，打开自己的内心，拥抱真正的人生。

文笔真的很烂了，我自己能感觉到，这就是长期只读专业书的后果，真的很沮丧，呵呵。

不过我相信还来得及，只要能及时转变观念，善于思考和反省，勤于实践，一切都会好起来！

相信自己！To be a better man.

SunnyJay 2016年5月26日
